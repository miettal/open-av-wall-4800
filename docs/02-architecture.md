# Architecture

## System Overview

```
Open AV Wall

├── Structural Frame
│
├── Display Modules
│
├── Camera Modules
│
├── Power Distribution
│
├── Network Distribution
│
├── Cable Management
│
└── Future Expansion
```

---

# Structural Modules

The wall consists of three structural modules.

```
1500
1800
1500
```

The center module acts as the primary load-bearing and service area.

---

# Display Layer

Displays are mounted using independent VESA modules.

Display positions may change without redesigning the structural frame.

---

# Infrastructure Layer

Behind the displays, space is reserved for:

* Power
* Ethernet
* HDMI
* USB
* PoE
* Future wiring

---

# Mechanical Design Rules

Primary structure

* 50x50 aluminum extrusion (tentative)

Secondary members

* 40x40 aluminum extrusion

Accessory rails

* 30x30 aluminum extrusion

---

# Coordinate System

Origin:

Center of floor.

```
X

Width

Y

Depth

Z

Height
```

All future CAD shall use this coordinate system.

---

# Design Principle

The structural frame is permanent.

Displays are consumable modules.

The platform should survive multiple generations of displays.
